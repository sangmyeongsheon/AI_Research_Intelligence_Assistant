# LabTrace 발표 진행

1. `RECORDING_SCRIPT.md`를 읽어 `transfer-handover.mp3`를 준비합니다.
2. 별도 제공된 업로드 파일 ZIP의 압축을 풉니다.
3. Overview에서 `새 프로토콜 만들기`를 선택합니다.
4. 녹음 파일과 이미지, PDF, TXT, Markdown 파일을 한 번에 업로드합니다.
5. 작성자, 날짜와 자료 성격을 확인한 뒤 분석을 시작합니다.
6. 인식된 원문과 메타데이터를 짧게 확인합니다.
7. 조건 충돌과 누락 정보를 검토합니다.
8. 생성된 프로토콜에서 각 단계의 원본 근거를 엽니다.
9. 상태를 확정하고 Markdown 또는 PDF로 내보냅니다.

발표에서는 서로 다른 매체를 한 번에 구조화하는 과정, 충돌을 임의로 숨기지 않는 점, 각 조건에서 원본으로 되돌아갈 수 있는 점을 중심으로 보여줍니다.
