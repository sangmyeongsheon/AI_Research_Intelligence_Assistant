# ARIA 확정 데모 파일

다음 5개 파일을 `새 프로토콜 만들기` 화면에 한 번에 첨부합니다.

1. `transfer-handover-recording.m4a` — 실제 음성 인수인계
2. `handwritten-transfer-note.png` — 수기 실험 노트
3. `neural-systems-lab-reference-protocol.pdf` — 기존 참고 프로토콜
4. `troubleshooting-experience.md` — 문제 해결 경험 기록
5. `latest-direct-note.txt` — 최신 직접 지시 메모

분석 후에는 각 출처의 인식 결과를 확인하고, 충돌 조건과 누락 질문을 검토한
다음 생성된 프로토콜을 엽니다.
